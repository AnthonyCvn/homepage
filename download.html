Reduct Storage supports Linux, macOs and Windows on AMD64 platform.

  • Linux
  • macOs
  • Windows

Reduct Storage

Binary

    wget https://github.com/reduct-storage/reduct-storage/releases/download/{{ site.projects.reduct_storage.version }}/reduct-storage.linux-amd64.tar.gz
    tar xfv reduct-storage.linux-amd64.tar.gz
    RS_DATA_PATH=./data ./reduct-storage

Docker

    docker run -p 8383:8383 -v ${PWD}/data:/data reductstorage/engine:latest
